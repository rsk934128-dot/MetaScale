# **App Name**: MetaScale

## Core Features:

- Hierarchical Campaign Manager: Visualize and manage the entire Meta ad structure including Campaigns, Ad Sets, and Ads in a nested dashboard view.
- AI Ad Copywriter Tool: Utilizes a generative AI tool to draft compelling ad headlines and body text based on your campaign objectives and target audience segments.
- Targeting Intelligence Tool: An AI-powered reasoning tool that suggests optimized audience demographics and budget allocations to maximize performance across different platforms.
- Creative Asset Library: A centralized UI repository for managing ad creatives, ensuring easy reuse of visual elements across multiple ad campaigns.
- Performance Analytics Dashboard: Direct integration with the Marketing API to display real-time insights, analytics, and ROI data for every level of your advertising structure.
- Automated Deployment Engine: Synchronize campaign updates, budget changes, and scheduling directly to the Meta ad account through API-driven automation.

## Style Guidelines:

- The palette is informed by precision and high-performance technology. We utilize a professional deep blue (#5c8aff) as the primary color to inspire confidence and authority.
- A sophisticated dark mode background (#1a1c23) is employed to minimize eyestrain and provide a sleek, focused workspace for data analysis.
- An electric cyan accent (#0de2f2) is used for call-to-action elements and real-time status indicators, providing high contrast and visibility.
- A pairing of 'Space Grotesk' for bold, tech-forward headlines and 'Inter' for highly readable body text is recommended to balance data density with clarity.
- Linear, minimalist icons using varying stroke weights to differentiate between Campaign, Ad Set, and Ad Creative levels.
- A clean, data-intensive grid layout that emphasizes the hierarchical relationship of Meta objects using consistent horizontal alignment.
- Micro-interactions such as progressive loading state transitions when fetching API data and subtle scale-ups for creative previews.
