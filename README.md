## 🛠 Tech Stack

*   **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
*   **AI Engine:** [Genkit](https://firebase.google.com/docs/genkit) (Google AI/Gemini)
*   **Backend:** [Firebase](https://firebase.google.com/) (Auth, Firestore)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
*   **Analytics:** Recharts for real-time telemetry visualization.

## 🛠 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-repo/sovereign-os.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Setup Environment Variables:** Create a `.env` file with your Firebase and Google AI credentials.
4.  **Run Development Server:**
    ```bash
    npm run dev
    ```
